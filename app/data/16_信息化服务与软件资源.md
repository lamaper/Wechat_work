# 信息化服务与软件资源（BIT 新生）

> 所有信息化服务以信息化服务部门 https://itc.bit.edu.cn/ 公告为准。正版软件授权以学校正版软件平台和厂商授权协议为准。

## 一、统一身份认证

统一身份认证（SSO）是你访问所有校内信息系统的**唯一通行证**。

1. 账号：学号。
2. 初始密码：以录取通知书、迎新网（https://hi.bit.edu.cn/）或信息化服务部门通知为准。**首次登录后必须修改密码**（部分系统会强制跳转到密码修改页面）。
3. 密码安全建议：长度至少 8 位，包含大写字母、小写字母、数字和特殊字符（如 !@#$%）至少三类。不与微信、QQ、邮箱等任何第三方平台共用密码。建议每 3-6 个月更换一次。
4. 应用范围：教务系统、智慧北理（https://ehall.bit.edu.cn/）、校园网认证、图书馆系统（https://lib.bit.edu.cn/）、VPN/WebVPN（https://webvpn.bit.edu.cn/）、校园邮箱、在线教学平台、正版软件平台、财务系统等，几乎所有校内在线服务统一使用本账号登录。

### Q: 密码忘了怎么找回？
A: 方式一（自助）：在统一身份认证登录页面（https://sso.bit.edu.cn/）点击"忘记密码" → 通过绑定的手机号接收短信验证码或通过绑定的密保邮箱接收重置链接 → 设置新密码。方式二（人工）：携带本人学生证或身份证到信息化服务大厅（位置见 https://itc.bit.edu.cn/）现场办理密码重置。**建议在首次登录后立即绑定手机号和密保邮箱，避免后续遗忘密码时无法自助找回。**

### Q: 账号被锁定了怎么办？
A: 多次输入错误密码（通常连续 5-10 次）可能触发账号临时锁定保护。一般等待 10-30 分钟后自动解锁即可重试。如长时间未自动解锁或急需使用，联系信息化服务部门（https://itc.bit.edu.cn/）处理。

## 二、校园网络

### 连接与认证

1. 在校园内搜索无线网络 Wi-Fi 信号（SSID 通常为 BIT、BIT-Web 或类似名称，以实际搜索到的为准）。
2. 连接后打开浏览器访问任意网页，系统通常会自动跳转到网络认证页面（Portal 页面）。如不跳转，手动访问认证门户地址（一般会通过迎新指引或信息化服务网站告知）。
3. 在认证页面输入统一身份认证账号和密码 → 点击登录 → 认证成功后即可正常上网。
4. 认证有效期：一般为 24 小时或更长，到期后需重新登录认证。部分设备和系统可设置"无感认证"（首次登录后一段时间内自动连接无需重复登录），以信息化服务部门提供的最新功能为准。
5. 宿舍区部分楼栋可能提供有线网络接口——将网线连接电脑和墙上的网口，其余步骤与无线认证相同。

### Q: 校园网怎么计费？
A: 校园网为在校生提供一定额度的免费流量（如每月 20GB、50GB 等，具体数值以当年度公告为准），超出免费额度后按流量计费。计费政策、套餐选项和缴费方式以信息化服务部门（https://itc.bit.edu.cn/）的"网络服务"栏目公告为准。属于教学和科研使用的部分资源（如图书馆数据库、教务系统等校内站点）的访问一般不计入套餐流量。

### Q: 校园网速度怎么样？
A: 在正常时段（非高峰时段）可满足视频学习、资料下载、在线会议等日常需求。高峰时段（晚间 20:00-23:00 学生集中上网时段、选课开放日等）由于同时在线用户数极多，网速感知可能下降。这不是你个人的网络问题，而是共享带宽在高并发下的客观表现。

### Q: 宿舍能自己装路由器或拉宽带吗？
A: 以信息化服务部门和宿管规定为准。部分宿舍楼允许设置个人无线路由器（需注意不干扰校园网正常运行、仅限本人使用、注意用电安全和网络安全）。私自拉接运营商宽带一般不被允许——校园网络由学校统一规划和管理。具体可咨询宿管和信息化服务部门（https://itc.bit.edu.cn/）。

## 三、VPN 与校外访问

### Q: 在校外怎么访问图书馆数据库（中国知网等）？
A: 通过学校 WebVPN（https://webvpn.bit.edu.cn/）：
1. 访问 https://webvpn.bit.edu.cn/
2. 使用统一身份认证账号和密码登录。
3. 在 VPN 门户页面中找到"图书馆数据库"或直接在页面内的地址栏中输入目标数据库网址（如 www.cnki.net）。
4. 通过 WebVPN 通道访问，系统会自动将你识别为校内用户，从而可以下载学校已购买的全文文献。
5. 除 WebVPN（网页版，无需安装客户端）外，学校也提供客户端版 VPN（需在校内时从 https://itc.bit.edu.cn/ 下载和安装配置），客户端版提供更稳定和全面的连接，适合需要长时间访问校内资源的场景（如远程桌面、大文件传输等）。

### Q: VPN 连不上或速度很慢怎么办？
A: 1) 检查统一身份认证是否正常。2) WebVPN 在高峰期（如期末论文季）访问量激增可能变慢，可错峰使用。3) 尝试切换为客户端版 VPN（通常比 WebVPN 更稳定）。4) 检查本地网络（切换 Wi-Fi 或用手机热点测试）。若问题持续，联系 https://itc.bit.edu.cn/。

## 四、校园邮箱

### Q: 学生有学校邮箱吗？怎么开通？
A: 有。北京理工大学为在校生提供以 @bit.edu.cn 为域名的校园邮箱。开通方式以信息化服务部门（https://itc.bit.edu.cn/）通知为准——一般通过统一身份认证后自助激活开通。建议所有同学在入学后尽快开通并定期查看。

### Q: 校园邮箱有什么用？为什么建议开通？
A: 校园邮箱的重要性远不止"又一个邮箱"：
- 学校发送的正式通知（教务变动、缴费提醒、重要活动等）会发送到校园邮箱。
- 联系导师、申请科研项目、注册学术会议时使用 @bit.edu.cn 邮箱比使用 QQ 邮箱更专业可信。
- **大量面向学生的优质资源需要 .edu 邮箱验证**：GitHub Student Developer Pack（包含 GitHub Pro、GitHub Copilot 免费使用权、域名、云服务抵扣券等价值数千美元的资源）、JetBrains 全家桶教育授权、AutoDesk 教育版、Notion 教育版、Spotify 学生优惠等。没有 .edu 邮箱意味着大量这些优质免费资源与你无缘。
- 毕业离校后邮箱通常仍可保留使用一段时间（或转为校友邮箱），是永久的 BIT 身份标识。

## 五、正版软件资源

学校为在校生购买或提供了多种正版软件授权：

### 操作系统与办公软件
- **Windows 操作系统教育版**：通过学校正版软件管理与服务平台下载 ISO 镜像和获取激活码。一般提供 Windows 10/11 教育版。
- **Microsoft Office 365 教育版**：包含 Word、Excel、PowerPoint、OneNote、OneDrive 云存储等。通过学校正版软件平台激活后，在校期间可持续使用。

### 开发与科研工具
- **JetBrains 全家桶**（IntelliJ IDEA、PyCharm、CLion、WebStorm 等）：凭 @bit.edu.cn 校园邮箱在 JetBrains 官网申请学生授权（https://www.jetbrains.com/student/），在学期间每年免费续期。
- **MATLAB**：学校是否购买了 Campus License 取决于当年采购情况。如有，在校生可通过校园邮箱和校园网免费获取和激活。具体查阅信息化服务（https://itc.bit.edu.cn/）或数学与统计学院相关通知。
- **Visual Studio**：Microsoft 为教育用户提供的免费版本可通过学校正版软件平台或 Microsoft 教育计划获取。
- 其他科研软件（Mathematica、Origin、SPSS、Ansys 等）：视学校各学院的采购和授权情况而异，详情查询信息化服务网站或学院通知。

### Q: 怎么获取免费的 Office 和 Windows？
A: 通过学校正版软件管理与服务平台下载和激活。平台网址和使用指南以信息化服务部门 https://itc.bit.edu.cn/ 公告为准。一般流程：使用统一身份认证登录正版软件平台 → 查看可获取的软件列表 → 下载所需软件和激活工具 → 按指南完成安装和激活。**请从学校官方平台下载，不要从第三方网站搜索"激活工具"（此类工具常捆绑恶意软件）。**

### Q: GitHub Student Pack 怎么申请？
A: 1) 先开通校园邮箱（@bit.edu.cn）。2) 访问 https://education.github.com/pack 用校园邮箱注册或关联你的 GitHub 账号。3) 按提示提交学生身份验证（可能需要上传学生证照片或学校 Portal 截图）。4) 审核通过后即可享受 GitHub Pro、GitHub Copilot 免费使用、域名（.me 或 .tech）、DigitalOcean / Namecheap 等云服务抵扣券在内的众多学生专属资源。

## 六、在线教学与学习平台

1. **在线教学平台**（如课堂派、超星学习通、中国大学 MOOC 学校专版等）：用于教师发布课件、布置和提交作业、在线测试、课堂互动和课程直播/录播。各课程使用的具体平台由任课教师选定并第一节课告知学生。一般使用统一身份认证登录或通过教师提供的课程邀请码加入。
2. **智慧北理**（https://ehall.bit.edu.cn/）：统一网上办事服务大厅，集成了教务服务、学工服务、财务缴费、一卡通服务、图书服务、网络服务、离校服务等，不必逐一访问各系统。
3. **图书馆电子资源**（https://lib.bit.edu.cn/）：中英文数据库、电子图书、学位论文、多媒体学习资源等。

### Q: 在线课程平台怎么用？
A: 开学第一周各课教师会在课堂上说明本课程使用的在线平台、加入方式和主要功能（交作业用哪个、看课件用哪个、在线讨论用哪个）。按照教师指引操作即可。如果不确定，可以向同班同学或课程助教确认。

## 八、回答边界

- 校园网带宽、流量策略、资费套餐和认证技术方案可能随学校信息化建设和运营商合作情况调整，以 https://itc.bit.edu.cn/ 最新公告为准。
- 正版软件的具体授权范围（版本、功能模块、使用期限）以学校与厂商签订的采购合同及学校正版软件管理平台的操作指南为准。毕业后授权通常终止。
- 第三方教育优惠（GitHub、JetBrains 等）的申请条件和福利内容由各厂商自主决定和调整，学校无法控制。
- 在线教学平台的具体使用由各任课教师决定和通知，知识库仅提供通用介绍。
- 系统故障、账号异常等具体技术问题的排查和恢复以 https://itc.bit.edu.cn/ 技术支持为准。
