# 北理工就业与知名校友

> 版本：2026-05-20
> 说明：本文件涵盖毕业生就业去向、薪资水平、知名校友和杰出人物等信息。

## 2024届毕业生基本情况

| 指标 | 数据 |
|------|------|
| 毕业生总数 | 3,680人 |
| 整体毕业去向落实率 | 98%（居国内高校前列） |
| 总体深造率 | 71.90% |
| 国内升学 | 62.93%（其中考研率约43%） |
| 出国(境)深造 | 8.97% |
| 直接就业比例 | 约28.10% |

## 薪资水平

- 应届生平均工资：约6,300元/月（较全国平均高约1,300元）
- 计算机、电子信息等热门专业实际薪资明显高于平均水平

## 就业行业与单位

### 主要行业分布
IT/软件/互联网、金融投资、汽车/装备制造、航空航天、国防军工、信息通信。

### 重点就业单位
| 类别 | 单位 |
|------|------|
| 航空航天 | 中国航天科技集团、中国航天科工集团、中国航空工业集团 |
| 国防军工 | 中国兵器工业集团、中国电子科技集团、中国工程物理研究院 |
| 能源/电力 | 国家电网、中国卫星网络集团 |
| 科技/互联网 | 华为、字节跳动、百度、阿里巴巴、腾讯、京东、小米 |
| 汽车 | 比亚迪、北汽集团 |
| 金融 | 工商银行、农业银行、中国银行、中信集团 |
| 通信 | 中国移动 |

### 就业单位性质
- 超2/3毕业生进入大型央企、国企和世界500强企业
- 超75%毕业生到国家重点单位及科技前沿领域就业
- 到中央国家机关和地方政府就业人数年均增长超25%，定向选调生年均增长超30%

## 就业地域分布
主要流向：北京、上海、广东、天津、重庆、浙江、江苏、山东、四川。

## 就业满意度
- 毕业生对落实工作满意度：超97%
- 用人单位对毕业生满意度：超97%
- 七成毕业生通过学校渠道落实就业单位
