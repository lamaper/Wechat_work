## 一、BIT101 综合校园平台

### 项目概述
BIT101 是北理工学生自建的**非官方综合校园服务平台**，GitHub 组织 BIT101-dev 维护。项目自述为"针对北京理工大学搞的怪东西"，是校园内最知名的学生技术项目之一。

### 主要功能
- **课程表同步**：自动从教务系统同步课表
- **乐学日程同步**：自动从乐学（BIT Moodle）拉取日程
- **校园地图**：提供校园地图浏览
- **成绩查询**：配合BIT101接口快速查询学业成绩，支持详细分析和综合分析
- **考试日程**：查看考试安排，支持一键导入系统日历
- **课程评价与讨论**：学生可在平台上评价课程和老师，支持reaction/comments互动
- **社区讨论**（话廊）：校内论坛，自由交流
- **消息聚合**：订阅多种通知，掌握全校消息
- **推荐系统**：通过Meilisearch+Gorse实现课程搜索和个性化推荐

### 技术架构

| 组件 | 技术 | 仓库 |
|------|------|------|
| 前端 | Vue | [BIT101-dev/BIT101](https://github.com/BIT101-dev/BIT101) |
| 后端 | Go（原Python重写） | [BIT101-dev/BIT101-GO](https://github.com/BIT101-dev/BIT101-GO) |
| Android | Kotlin | [BIT101-dev/BIT101-Android](https://github.com/BIT101-dev/BIT101-Android) |
| 数据库 | PostgreSQL | - |
| 缓存 | Redis | - |
| 搜索 | Meilisearch | - |
| 推荐 | Gorse | - |

### 性能
Go后端相比原Python后端，接口并发性能提升约**15倍**（从22.71 req/s → 340.31 req/s）。

### 项目地址
- 主站：https://bit101.cn
- 前端：https://github.com/BIT101-dev/BIT101
- 后端：https://github.com/BIT101-dev/BIT101-GO
- Android：https://github.com/BIT101-dev/BIT101-Android

## 二、BITNP 网络开拓者协会项目

网络开拓者协会（NetPioneer Association，简称网协/BITNP）成立于1997年，是北理工历史最悠久的学生技术社团之一，GitHub上共有64个开源仓库。

### 核心项目列表

### 1. BIThesis — LaTeX学位论文模板
- **Stars**：⭐ 667（BIT相关项目最高）
- **语言**：TeX
- **许可证**：LPPL-1.3c
- **功能**：提供本科毕业设计、硕士/博士学位论文的LaTeX模板
- **特色**：
  - 符合北理工论文格式规范
  - 支持本科、硕士、博士三种学位类型
  - 包含完整的中文文档和使用说明
  - 提供Overleaf在线编辑支持
- **仓库**：https://github.com/BITNP/BIThesis
- **文档**：https://bithesis.bitnp.net

### 2. bitsrun — 校园网命令行登录工具
- **Stars**：⭐ 75+
- **语言**：Python
- **许可证**：WTFPL
- **功能**：北理工10.0.0.55校园网登录/登出的命令行工具
- **特色**：支持CLI和Raycast脚本，配置文件管理

### 3. NPLive — 校园活动直播平台
- **功能**：免流量校园直播平台
- **技术**：基于SRS/RTMP，Docker部署
- **场景**：深秋歌会等大型校园活动直播

### 4. 北理镜像站
- **网址**：https://mirror.bit.edu.cn
- **功能**：开源软件镜像服务（Linux发行版、Python包、Docker镜像等）
- **相关仓库**：bit-mirror-sync（镜像同步脚本）

### 5. 电脑诊所
- **功能**：电脑义诊预约管理系统
- **服务**：免费为全校师生提供电脑维护

### 6. 国防知识竞赛系统
- **功能**：在线答题竞赛平台
- **仓库**：contest-bitnp

### BITNP相关链接
- **GitHub组织**：https://github.com/BITNP（64个仓库）
- **官方B站账号**：参与深秋歌会等技术转播

## 九、北理工开源生态总结

### 数据概览
- GitHub上与BIT直接相关的开源仓库**100+**
- 最大组织：BITNP（64个仓库）
- Star最高项目：BIThesis（667 stars）
- 最活跃领域：校园网登录工具（10+种语言实现）
- 最丰富生态：Greasy Fork用户脚本（20+个活跃脚本）

### 生态特点
1. **实用主义**：几乎每个工具都解决一个真实的校园痛点
2. **多语言实现**：同一需求（如校园网登录）有Python、Go、Rust、Swift等多种实现
3. **持续维护**：BIThesis、best-of-bits等核心项目持续更新
4. **社区活跃**：Greasy Fork上BIT专属脚本的数量和质量在全国高校中名列前茅
5. **技术多样性**：Vue、Go、Rust、Kotlin、SwiftUI、Deno等技术栈均有涉及

### 对学生的意义
- 选课不再纯靠运气（有抢课脚本和饱和度分析）
- 论文排版不再痛苦（BIThesis一键搞定）
- 评教不再繁琐（一键评教）
- 校园网登录不再麻烦（命令行一键登录/登出）
- 成绩分析不再手动（自动导出算GPA）
- 课程学习有资料库（BIT-CS、BIT-AI-Review等）
