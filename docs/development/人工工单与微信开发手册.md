# 人工工单与微信开发手册

## 1. 相关模块

- `app/backend/modules/feature3/service.py`
- `app/backend/modules/feature3/routes.py`
- `app/backend/modules/feature3/wechat_utils.py`

## 2. 工单主流程

1. 新生提交问题
2. 问题进入待处理列表
3. 学长提交答复
4. 用户通过网页或查询码查看结果
5. 问题关闭

状态流转：`pending -> answered -> closed`

## 3. 微信链路

微信公众号消息入口：`/wechat`

支持能力：

- 接收文本消息
- 解析查询命令
- 回传工单结果
- 可选客服消息推送

## 4. 关键配置

- `WECHAT_TOKEN`
- `WECHAT_APPID`
- `WECHAT_APPSECRET`
- `FEATURE3_ENABLE_WORKER`
- `FEATURE3_ENABLE_WECHAT_PUSH`

## 5. 开发注意

- 修改查询码逻辑时，保持兼容已有记录
- 修改 worker 行为时，注意重试和幂等
- 微信相关改动要在测试号环境先验证
