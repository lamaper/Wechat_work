# 前端与页面开发手册

## 1. 页面结构

前端页面全部位于 `app/web/`：

- `index.html`
- `login.html`
- `chat.html`
- `place.html`
- `freshman.html`
- `senior.html`

样式主文件：`theme.css`
公共脚本：`common.js`

## 2. 路由映射

页面访问路由由 `app/backend/modules/pages/routes.py` 统一注册。

改页面时需要同时确认：

- 页面文件本身
- 路由是否可达
- 登录拦截是否符合预期

## 3. 开发约束

- 尽量复用 `theme.css` 现有样式变量和组件风格
- 不在页面里硬编码后端地址，统一走同源 `/api/...`
- 修改交互后至少验证一次移动端视口

## 4. 地图页面注意项

`place.html` 依赖腾讯地图 Key。未配置 Key 时页面可打开，但真实检索不可用。

## 5. 回归检查清单

1. 登录页能正常打开
2. 登录后能进入首页
3. chat/place/freshman/senior 四页可访问
4. 页面内 API 调用返回符合预期
