# 四个功能代码框架详解

## 1. 登录与权限

### 代码位置

- `app/backend/modules/auth/service.py`
- `app/backend/modules/auth/routes.py`

### 核心职责

- 登录认证
- Session 构建
- 权限位控制
- 页面与 API 访问拦截

## 2. AI 问答

### 代码位置

- `app/backend/modules/ai/service.py`
- `app/backend/modules/ai/kb.py`
- `app/backend/modules/ai/web_fetcher.py`
- `app/backend/modules/ai/routes.py`

### 核心职责

- 对问题做分流
- 检索本地知识库
- 补充网页证据
- 调用 LLM 汇总

## 3. 地图功能

### 代码位置

- 前端页面：`app/web/place.html`
- 公共脚本：`app/web/common.js`
- 页面入口：`app/backend/modules/pages/routes.py`

### 核心职责

- 地点检索和坐标展示
- 校内地点快速查看

## 4. 人工工单与微信

### 代码位置

- `app/backend/modules/feature3/service.py`
- `app/backend/modules/feature3/routes.py`
- `app/backend/modules/feature3/wechat_utils.py`

### 核心职责

- 新生提问
- 学长答复
- 查询码追踪
- 微信消息回调与推送
