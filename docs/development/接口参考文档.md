# 接口参考文档

## 说明

这里只列当前常用接口分组和用途，详细请求体以代码实现为准。

## 1. 认证接口

- `POST /api/auth/login`：登录
- `POST /api/auth/logout`：退出
- `GET /api/auth/me`：当前登录信息

## 2. AI 接口

- `POST /api/chat`：问答主接口
- `POST /api/rebuild_index`：重建知识库索引
- `GET /api/kb_query`：知识库检索调试
- `GET /api/demo_status`：运行状态诊断

## 3. 工单接口

- `POST /api/questions`：提交问题
- `GET /api/questions/mine`：我的问题
- `GET /api/questions/track`：按编号+查询码追踪
- `PUT /api/questions/<id>/close`：关闭问题
- `GET /api/tasks/pending`：待处理
- `GET /api/tasks/answered`：已答复
- `POST /api/answers`：提交答复
- `GET /api/feature3_status`：工单模块状态

## 4. 微信回调

- `GET /wechat`：回调校验
- `POST /wechat`：接收公众号消息

## 5. 页面路由

- `/`
- `/login`
- `/chat`
- `/place`
- `/freshman`
