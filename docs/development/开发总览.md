# 开发总览

## 1. 适用范围

本文档用于当前仓库的日常开发和维护，默认基于 `app/backend/app.py` 这套启动方式。

## 2. 开发入口

- 后端代码：`app/backend/modules/`
- 前端页面：`app/web/`
- 知识库数据：`app/data/`
- 自动化脚本：`app/backend/tests/`

## 3. 基本开发流程

1. 在 `app/backend` 配置 `.env`
2. 启动服务并验证页面可访问
3. 修改功能代码
4. 跑冒烟测试
5. 做页面回归

## 4. 常用命令

```powershell
Set-Location .\app\backend
conda activate D:\envs\wechat_work
D:\envs\wechat_work\python.exe .\app.py
D:\envs\wechat_work\python.exe .\tests\feature3_smoke_test.py
```
