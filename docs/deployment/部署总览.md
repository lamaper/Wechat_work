# 部署总览

## 1. 部署形态

### 本地开发

- Python 直接启动 `app.py`
- 适合联调和日常开发

### Linux 生产

- Nginx + Gunicorn + systemd
- 适合公网稳定运行

## 2. 必备条件

- Python 3.10+
- backend 目录可读写
- `.env` 配置完整
- 地图 Key、LLM Key、微信配置按需提供

## 3. 关键文件

- `app/backend/app.py`
- `app/backend/.env`
- `app/backend/requirements*.txt`
- `deploy/nginx/wechat_assistant.conf`
- `deploy/linux/systemd-service.md`
- `deploy/windows/nssm-service.md`
